# DBMS project

## quickstart
+ linux
```
python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
docker compose up -d
```

+ windows
```
python -m venv .venv
.venv/Scripts/activate
pip install -r requirements.txt
docker compose up -d
```

The user can also manually setup by just running the sql files in their own postgresql instance, through cli or tools like pgadmin

## prerequisites

+ python
+ for windows, WSL enabled, docker installed
+ for linux, see docker setup guide: https://docs.docker.com/engine/install/linux-postinstall/
